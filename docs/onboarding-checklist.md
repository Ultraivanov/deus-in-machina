# Onboarding Checklist

## Buildrail — New User Steps

1. Install dependencies: `npm install`.
2. Start the MCP server: `npm run dev`.
3. Initialize a project with `initialize_project`.
4. Review `get_next_step` output and confirm scope.
5. Approve a Change Plan and run `start_session`.
6. Submit results with `submit_agent_result`.
7. Run `validate_scope`, then `explain_changes`.
8. Complete with `complete_task`.
9. Inspect state in `.assistant/` for progress and history.
