## Block Goal

Enhance the ruleset system with configurable rule packs, severity levels, and flexible validation policies to support different project requirements and compliance levels.

## Definition of Done

- Rule packs organized by category (tokens, layout, accessibility, etc.)
- Severity levels configurable per rule (error, warning, info, ignore)
- Rule enable/disable configuration
- Preset rule packs (strict, relaxed, minimal)
- Custom rule pack creation support
- Validation engine uses severity from ruleset
- Documentation for rule pack configuration
