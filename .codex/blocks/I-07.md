## Block Goal

Implement full bidirectional sync between Figma Variables and DTCG-compliant JSON tokens, enabling export (Figma → JSON) and import (JSON → Figma) with full fidelity.

## Definition of Done

- Export: Figma variables → DTCG JSON with all types, modes, aliases
- Import: DTCG JSON → Figma variables (creates/updates collections, modes, variables)
- CLI commands: `export-variables`, `import-variables`, `sync-variables`
- Support: colors (hex, rgba, hsla), numbers, strings, booleans, aliases
- Multi-mode support (light/dark/custom themes)

### I-07-T1 + T2 Change Plan (Approved & Executed)

Created Figma bidirectional sync module:
- `src/figma/exporter.js` — Export Figma variables to DTCG JSON
  - Supports: COLOR, FLOAT, STRING, BOOLEAN types
  - Color formats: hex, rgba-css, rgba-object, hsla-css, hsla-object
  - Multi-mode support via $extensions.mode
  - Alias resolution (VARIABLE_ALIAS → {path})
  - DTCG format ($value, $type) or standard (value, type)
  
- `src/figma/importer.js` — Import DTCG JSON to Figma variables
  - Creates/updates variable collections
  - Creates/updates variables with values
  - Two-pass alias resolution
  - Multi-mode value assignment
  - Dry-run mode for preview

- `src/figma/index.js` — Module exports
