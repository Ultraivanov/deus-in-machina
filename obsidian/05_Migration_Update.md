# Миграция и обновления

## Режимы

- `new` — чистая инициализация.
- `legacy` — перенос в существующий проект.
- `upgrade` — апгрейд версии фреймворка.

## Обновления

- Проверка версии в `start`.
- При наличии апдейтера обновление применяется автоматически.
- Пакет обновления касается только Codex‑части.
